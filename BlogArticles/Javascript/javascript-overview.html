对Javascript语言特性的认识

姚川松 — 写于 2015/3/21

Javascript是运行在浏览器中的Javascript引擎上的脚本语言，它是一种基于原型的面向对象的语言。浏览器厂商都采用Javascript实现了各自浏览器的DOM和BOM，也就是说浏览器中面向程序员最重要的那一部分是用Javascript写的。

Javascript的语言特性主要是：宽松数据类型；单线程的程序执行方式；基于原型

宽松类型

也就是官方说的动态类型，在Javascript中定义一个变量是不需要指定其具体类型的，这和C++，Java这样的语言有很大的不同。

单线程的执行方式

在对Javascript代码进行自动化测试时，经常会用到setTimeout函数来完成测试过程。

基于原型的面向对象

Javascript任然是面向对象的语言，虽然在使用过程中并没有像C++和Java那样通过类去创建然后使用对象。Javascript通过Prototype实现面向对象的封装和继承。
