什么是闭包?

在编程语言中, 闭包的概念是内部类, 内部类可以访问外部类的私有数据, 在Javascript中, 闭包就是函数, 在闭包中可以访问到定义该闭包的函数的数据域.

闭包在Javascript中的应用有哪些?

在Javascript中, 函数就是闭包. 所以, Javascript中闭包的应用就有回调函数, 类的实现, 模块化

什么是JSON

JSON的全称是Javascript Object Notation, 是一种数据表示格式. 可以表示的数据包括Number, String, Boolean, Array, 对象(大括号), null

什么是this

就是通过new关键字创建的对象. 在通过new关键字调用函数时, 函数中的this会有初始值, 包括constructor, prototype上属性的引用, __proto__, 在函数的最后会返回this作为对象实例.

什么是arguments

arguments是函数的参数对象, 它可以像数组那样被操作, 拥有length属性, 有下标操作, 但是它不是数组. arguments instanceof Array得到的false.

什么是ajax

ajax全称是Asynchronous Javascript And XML, 是一种前端的交互技术. 通过ajax实现了请求少量数据, 局部更新页面. 它通过XML表示HTTP请求, 并通过回调函数处理HTTP请求的响应.

ajax的核心技术

ajax的核心是XMLHTTPRequest对象, 即XHR对象.

事件绑定有哪些方法

浏览器提供的有,IE的attachEvent,标准的addEventListener. 另外jQuery还提供了bind, on, live, delegate四种方法.

不同事件绑定的方法有什么区别

attachEvent只支持冒泡事件, 不支持捕获事件. addEventListener方法支持冒泡事件和捕获事件, 可以通过第三个参数设置该事件是捕获事件(true)还是冒泡事件(false). 一般为了兼容IE, 将事件设置为冒泡事件.
