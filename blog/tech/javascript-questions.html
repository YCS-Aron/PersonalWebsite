Array.splice与slice

slice返回数组的一部分的拷贝, splice移除数组的一部分用于返回值. 注意他们的参数的含义不一样.


var array = [1, 2, 3, 4];
var copy = array.slice(1, 3);       //[2, 3] 第一个参数是开始的索引值, 第二个参数是结束的索引值,均从0开始
var removed = array.splice(1, 3);   //[2, 3, 4] 第一个参数是开始的索引值, 第二个参数是长度,均从0开始
console.log(array);                 //[1]
