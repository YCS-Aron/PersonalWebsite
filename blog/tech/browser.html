DOM树

DOM树, DOM的数据结构中保存了DOM相关的信息, 包括class, attributes等, 比如: 


struct Node {
    // data common to all nodes:
    children: Vec,

    // data specific to each node type:
    node_type: NodeType,
}

生成DOM树.

在解析HTML文本的过程中, 如果遇到script标签去加载远程JS文件或者style标签加载远程css文件, 那么会被阻塞, 即script的加载是同步的, 也就是为什么用jQuery实现JSONP的时候会使用同步请求.
